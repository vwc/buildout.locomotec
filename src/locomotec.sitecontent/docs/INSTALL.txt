locomotec.sitecontent Installation
----------------------------------

To install locomotec.sitecontent using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``locomotec.sitecontent`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        locomotec.sitecontent
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
